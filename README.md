# JustQuizzes
<img src="/assets/logo.png" width=64>

This is an application for making quizzes, playing with them, sharing with your friends!

## Release

- You can download the app [here](https://appdistribution.firebase.dev/i/4e8e662f20203b92)

- Demo video [here](https://drive.google.com/file/d/1R8OTszvSRrlgqH4qDzqJaVcOS-cByaVq/view?usp=sharing)

- Slide [here](https://docs.google.com/presentation/d/1sbUyGr60d9gUb1tmYMZqeiaNdH1oWQG5ZEEqy63KCxM/edit?usp=sharing)

## Built with

- Flutter
- Firebase

## Features

- Sign in/up with email
- Sign in with Google
- Reset password
- CRUD for quizzes
- CRUD for questions of each quiz
- Create quiz with voice by using speech to text library
- Share quiz
- Import quiz using token
